## 🚀 Featured Projects

### [LogSense AI — LLM-Powered Log Intelligence Platform](https://github.com/Dhevdharsan)
A production-grade containerized platform for AI-driven log anomaly detection and root cause analysis.

- 🔍 **Unsupervised ML pipeline** — TF-IDF → Isolation Forest → DBSCAN processes 500 logs in 1.5s, detecting 4.8% anomalies across 15 clusters
- 🧠 **RAG pipeline** — Llama 3 (Ollama) + pgvector embeddings generate plain-English root cause summaries from raw anomaly clusters
- ⚡ **Production architecture** — async FastAPI, Redis caching, Docker Compose with separated ingestion, retrieval, and LLM inference layers following microservices principles
- 📊 **Real-time dashboard** — Next.js frontend consuming streaming FastAPI endpoints

---

### [Agentic Research Assistant](https://github.com/Dhevdharsan)
A live AI agent that scrapes, synthesizes, and delivers hallucination-free research outputs in real time.

- 🤖 **Agentic AI** — OpenAI API with strict JSON-mode LLM extraction and source-URL attribution to prevent hallucinations
- ⚡ **Concurrent scraping** — ThreadPoolExecutor for parallel web retrieval with real-time FastAPI streaming
- 🚢 **Battle-tested deployment** — resolved Nginx load-balancer buffering (X-Accel-Buffering: no) and pydantic-core Rust compilation failures on Render; stable public deployment
