## 🚀 Featured Project

### [Smart Expense Analyzer](https://github.com/Nirupama-b/smart-expense-analyzer)
A full-stack AI-powered expense tracking application built as part of CS 520 at UMass Amherst.

- 📸 **OCR pipeline** — uploads receipt images, preprocesses with Pillow, extracts merchant, amount, and date using EasyOCR
- 🤖 **ML categorization** — zero-shot classification via facebook/bart-large-mnli maps raw OCR text to expense categories
- 📈 **XGBoost forecasting** — predicts next month's spend and computes burnout probability using lag features and a sigmoid function
- 💬 **Natural language queries** — ask questions about your expenses in plain English, powered by LangChain with a rule-based fallback
- ⚡ **Async pipeline** — Celery + Redis ensures OCR never blocks the UI; HTTP 202 returned in under 200ms consistently
- 🧪 **46 automated tests** — 63% backend coverage across unit and integration layers using pytest
